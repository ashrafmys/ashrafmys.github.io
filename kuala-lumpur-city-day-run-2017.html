Kuala Lumpur City Day Run 2017

Thu 02 March 2017
  • Event : Kuala Lumpur City Day Run 2017
  • Location : Padang Merbok route
  • Date : 26th Feb 2017
  • Start time : 730am
  • Weather : Raining in the beginning

Review

Somehow, I felt this was my worst outdoor running event but yet for some miracle, I did manage to get my best time for this event.

In my previous event, my average pace was 7’35” min/km. This new event however I managed to get 7’08” min/km

Some reflections on this run :

1. Running in the rain sucks. This was my first time running in the rain. It seems more energy was needed to run in this condition – to get rid of stressful feeling running wet, to carry water drenched clothes, to keep your face clear from rain water distraction while running

2. Hilly run. I was not expecting something like this – too many uphills. I walked for all the uphills. If I were to run again on this route, I might be mentally prepared on what should I do. My advice – for uphill run, try to continuously run (slow is OK) but if you can’t, do short bursts. But for downhill: don’t stop, keep running with long stride. This is the key to get better average pace.

3. Motivation to continuously run. During my previous run, the only thing that kept me running continuously was my desire to get the finisher medal and hence I must run within the time limit for the medal. I had no previous reference so I was so scared I might not make it which led to continuos run in the past event.

However, this time was different. I knew, no matter the completion time, I’ll get the medal. The only question was – can I finish within 1 hour 30 mins for E-Cert. To run within this time limitation was the only motivation I had.

Unfortunately, due to many uphills, I was demotivated and thought that I might not make it which led to – I couldn’t be bothered. This led to lots of walking compared to previous event.

I kept going though so that at least I was not that far off from the time limit. To my surprise, I did manage to run 12km within the time limit.

I knew I was more than capable to do better because during the final 1km, I could still run strongly with long stride.

Overall, it was a good experience to run on hilly Padang Merbok route.

Until next time !
