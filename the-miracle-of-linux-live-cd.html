The Miracle of Linux Live CD

Fri 20 January 2017

Problem:

I wiped clean my Linux partition on my hard disk. I never thought there would be any problem. To my horror, when I tried to boot my laptop, I couldn’t even get into my Windows at all. I received Grub Rescue prompt during booting up. Trying some tips from the Internet to no avail.

Solution:

  1. Luckily, there was a Ubuntu Mate live CD within reach.

  2. Changed boot order to boot from live CD

  3. From the command line, run boot-repair program but still no luck.

  4. Re-installed back Ubuntu Mate on my previous partition.

  5. Now, I could access my Windows back during booting up!

  6. This was not the best solution but still accomplished my objective to access my Windows. I didn’t have any Windows Repair media with me. Tried to create one but failed. Don’t know why.

Lessons Learnt:

  1. Do not simply erase Linux partition. You might accidentally erase Grub. Find out more about your Grub boot loader’s location before wiping your Linux partition.

  2. Make sure you have your Linux live CD with you. With this CD, you can still access the contents of your hard disk in case you couldn’t boot using your hard disk.
